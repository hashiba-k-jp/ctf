nmap – Network MAPper

posted 2024.02.05

Usage
% nmap [Options] {Target specification}

Options

  • -sT

    TCP connect scan (SYN, SYN-ACK, ACK, RESET)

  • -sS

    TCP SYN scan (SYN, SYN-ACK, RESET)

  • -sA

    TCP ACK scan (Only ACK flag -- can't distinguish whether ports are open or closed)

  • -sU

    UDP scan

  • -p [PORT RANGES]

    Port specification
    EX) -p22 (scan port 22), -p1-65535 (scan port from 1 to 65535)

  • -O

    Enables OS detection

  • -A

    enable OS detection along with other things (incl. versions)

  • -sV

    Enables version detection

  • --script[FILENAME|CATEGORY|DIRECTORY|EXPRESSSION]

    Runs a script scan
    --script=banner: banner scan
