C lang compiling

To set breakpoint by the file name : clang -g -O0 FILENAME.c [-o EXE_FILE]

Breakpoint

Set breakpoint by a func name
breakpoint set -n FUNC_NAME
Set breakpoint by a line number of the file
breakpoint set -f FILE_NAME -l LINE_NUM || b FILE_NAME:LINE_NUM
Show all the breakpoints
breakpoint list
Show delete breakpoints
breakpoint delete [BREAKPOINT_NUM, ...]
If the BP number(s) is not given, all BP will be deleted.

Run / Commands in running process

Run
r || run
(In running process) Run by step
s || step
(In running process) Run to next BP
c || continue
(In running process) Show values of variables
p VAR_NAME || print VAR_NAME
(In running process) Stop the process
kill
(In running process) Read the value of registers (REGISTER = rsp, rbp, ...)
register read REGISTER

Others

Run shell commands (ls, clear, etc...)
shell COMMANDS
