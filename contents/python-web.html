posted 2024.04.18

Before use...
import requests

Send Requests

  • Basic Expression

    [RESPONSE(optional)] = request.[METHOD]([URL], [PARAMETER(optional)])

  • Usual request

    e.g. r = requests.get(f'https://example.com/index.php?id={id}')

  • With parameter data

    e.g. s.post('https://example.com', data={'id': id, 'data': text})

Recieve response

Attributes of response object

Attr name Value
url url
status_code status code
headers response header
e.g. {'Content-Encoding': 'gzip', ...}
This attr is Type of CaseInsensitiveDict (basically same to dict)
text Contents of response (Decoded!)
e.g. <!DOCTYPE html> ...
content Contents of response (NOT Decoded; byte object)
