Structure

#main_content of all articles have the following structure.

<div class="col-md-9" id="main_content">
    <h2 class="mt-5">ARTICLE TITLE</h2>
    <p class="text-body-secondary" id="date">posted DATE, last update DATE</p>

    <div class="section">
        <a name="SECTION_NAME"></a>
        <h3>SECTION TITLE</h3>
        <p>PARAGRAPH</p>
        <p>PARAGRAPH</p>
        ... and other elements.
    </div>

    <div class="section">
        <a name="SECTION_NAME"></a>
        <h3>SECTION TITLE</h3>
        <p>PARAGRAPH</p>
        <p>PARAGRAPH</p>
        ... and other elements.
    </div>

    ... and other sections.
</div>

<div class="col-md-3">
    <div class="sticky-top mt-5" id="contents_list" style="top: 7rem;">
        <h5>Contents</h5>
        <p><a href="#SECTION_NAME" class="text-reset">SECTION TITLE</a></p>
        <p><a href="#SECTION_NAME" class="text-reset">SECTION TITLE</a></p>
        ... and other sections.
    </div>
</div>

The element of data <p> must have the class text-body-secondary and the id data.

Code

Use <code> for inline codes. In order to write code block, write as below (This is as same as the light content except for icon).

Title for code block
<div class="p-3 text-light-emphasis bg-light-subtle border border-light-subtle rounded-3 note">
    <h6><i class="bi bi-code-slash"></i> Helloworld.c</h6>
    <pre><code>Here is the code.</code></pre>
</div>

Here is an example.

Helloworld.c
#include <stdio.h>

int main(void){
    printf("Hello World!");
    return 0;
}
Warning
Inside of <pre>, "<" and ">" must be escaped to &lt; and &gt;.
In addition, the html file should not be indented. (Spaces of indent will be displayed.)
New lines just after <pre><code> and before </pre></code> is not needed.
