tcpdump

posted 2024.01.14

Note
tcpdump requires sudo.
Usage
% sudo tcpdump [EXPRESSIONS] [OPTIONS]

Expressions

The logical operators and and not are available.

  • host 192.168.XXX.XXX

    Ture when the source's and/or destination's IP address is 192.168.XXX.XXX.

  • dst host 192.168.YYY.YYY

    Ture when the destination IP address is 192.168.YYY.YYY.

  • src host 192.168.ZZZ.ZZZ

    Ture when the source IP address is 192.168.ZZZ.ZZZ.

  • port 443

    Ture when the source's and/or destination's port is 443.

  • src net 192.168.AAA.AAA mask 255.255.255.BBB

    Ture when the source network address is 192.168.AAA.AAA (subnet mask is 255.255.255.BBB).

Options

  • -W [FILENAME]

    Write out the result of capturing into the [FILENAME].

  • -p

    Show only the packets to current host.

  • -X

    Show ascii strings with hex string.
